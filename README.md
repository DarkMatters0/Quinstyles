# WMSU Uniform Shop

A website designed exclusively for WMSU students to purchase school uniforms or PE uniforms online. Students can conveniently order custom-made uniforms or select from various sizes: Small, Medium, and Large.

## About the Project

This project aims to simplify the process of buying school uniforms for WMSU students by providing an online platform where they can:
- Browse and purchase school or PE uniforms.
- Request custom-made uniforms tailored to their needs.
- Choose from standard sizes: Small, Medium, and Large.

### Built With

The project utilizes the following technologies:
- [HTML](https://developer.mozilla.org/en-US/docs/Web/HTML)
- [CSS](https://developer.mozilla.org/en-US/docs/Web/CSS)
- [JavaScript](https://developer.mozilla.org/en-US/docs/Web/JavaScript)
- [PHP](https://www.php.net/)
- [MySQL](https://www.mysql.com/)

## Getting Started

Follow these steps to set up the project locally.

### Prerequisites

Ensure you have the following installed:
```bash
# Example for PHP and MySQL
xampp
```

### Installation

Step-by-step guide to install and set up the project locally:
1. Clone the repository:
   ```bash
   git clone https://github.com/DarkMatters0/Quinstyles.git
   ```
2. Navigate to the project directory:
   ```bash
   cd your-repo
   ```
3. Start the XAMPP server and place the project folder in the `htdocs` directory.
4. Import the database file provided in the `db` folder into MySQL.

## Usage

Students can use the website to:
- Browse the available school and PE uniforms.
- Select their preferred size or request a custom-made uniform.
- Complete their purchase online.

To run the project:
```bash
# Open your web browser and navigate to
http://localhost/your-repo
```
